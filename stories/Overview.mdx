# Component Library Overview

Welcome to the PRISMqd UI Storybook. This catalog renders design primitives and dashboard widgets in isolation so you can iterate without running the full Next.js app.

## Running Storybook

```bash
bun run storybook
```

The command starts Storybook on port 6006. Pass a different port with `-- --port 7007` if 6006 is already in use.
